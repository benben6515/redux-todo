# 這是一個 react / redux 做的 Todo list

![](https://i.imgur.com/k5AepP7.gif)
[Demo](https://benben6515.github.io/redux-todo/)
### 功能
- 新增 todo
- 編輯 todo
- 刪除 todo
- 標記完成/未完成
- 清空 todo
- 篩選 todo（全部、未完成、已完成）

### 說明
- 使用技術：
  - create-react-app / redux
  - @reudxjs/toolkit
  - styled-components
- 有 2 個 Slice： filterSlice, todoSlice
- 試著玩了一點 3D 效果
